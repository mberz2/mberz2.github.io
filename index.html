[Projects]

[web and cloud security]


This project repository contains a variety of notebooks and Python scripts for addressing a variety of web and cloud security vulnerabilities including Cross-site Scripting, Cross-Site request forgery, and XXE/SQL injections. There are also five notebooks offering descriptions and remediation steps. The culminating project is a guided walkthrough on solving ten Business-Logic vulnerabity levels hosted on portswigger.net

[java/android app development]

This project repository contains a proof-of-concept/demonstration mobile application developed for Android in the Java programming language in using the Android Studio Development Environment. Using a combination of Java, XML, and Android widgets, this application allows for users to create and store "appointments" to internal device storage, view and search by a given owner or dates, and delete all. The project currently only runs on the emulator, but can be configured to run on any Android device.

[full-stack development]


This project demonstrates full-stack development using MongoDB, Express(.js), React(.js), Node(.js), collectively known as the MERN stack. In addition, the project makes use of Firebase for authentication. The application is limited in functionality but allows for login, send/receipt of messages, and chatroom creation. The project was bootstrapped with Create React App, using Redux and Redux Toolkit template. The project repository inclues installation instructions for localized testing and development.
